# ⚽ Smart Stadium AI

An AI-powered Smart Stadium Assistant built for the **Hack2Skill – Smart Stadiums & Tournament Operations Challenge**. This project leverages **Google Gemini AI** to improve stadium operations and enhance the fan experience during the FIFA World Cup 2026.

---

## 🚀 Features

### 🗺️ AI Smart Navigation
- Find the shortest route to seats, entrances, exits, washrooms, food stalls, parking, and emergency exits.
- Interactive digital stadium map.

### 👥 Real-Time Crowd Management
- Monitor crowd density.
- Receive congestion alerts.
- AI-powered route recommendations to avoid crowded areas.

### 🌍 Multilingual AI Assistant
- Supports multiple languages for international visitors.
- AI-powered conversations and assistance.

### ♿ Accessibility Assistance
- Wheelchair-friendly navigation.
- Accessible washrooms and elevators.
- Assistance point locator.

### 🏟️ Interactive Stadium Map
- Stadium layout with:
  - Entrances
  - Exits
  - Seating Sections
  - Washrooms
  - Accessible Washrooms
  - Food Courts
  - Food Stalls
  - Beverage Stalls
  - Parking Areas
  - First Aid Centers
  - Emergency Exits

## ⚙️ Installation

Clone the repository

```bash
git clone <repository-url>
cd smart-stadium-ai
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build the project

```bash
npm run build
```
